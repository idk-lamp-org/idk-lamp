# predictability-gate / docs

このディレクトリには、**predictability-gate**の背後にある運用ドキュメントが含まれています。

これらのファイルはチュートリアルや実装ガイドではありません。
実際の運用において、**いつAIを*使うべきではない*かをどのように判断するか**をドキュメント化しています。

## 読む順序

0. `00_failure_cases.md`  
   — なぜこのゲートが存在するのか（境界を必要とさせた失敗パターン）

1. `01_overview.md`  
   — このゲートとは何か、なぜ存在するのか

2. `02_gate_definition.md`  
   — G1〜G6の正式な定義

3. `03_templates.md`  
   — 現場ですぐ使えるテンプレート（ケースカードとゲートフォーム）

4. `04_scoring.md`  
   — スコアリングルールと交渉不可能なオーバーライド

5. `05_irreversibility.md`  
   — 不可逆的リスクはどのように判断されるか

6. `06_sop_numeric.md`  
   — 数値時系列ケースのためのSOP

7. `07_sop_non_structured.md`  
   — 非構造化データケースのための最小限のSOP

8. `08_consensus.md`  
   — 反対意見が出た場合の合意形成の方法

9. `09_handover_non_structured.md`  
   — 除外されたケースを引き継ぐための境界ノート

これらのドキュメントは意図的に最小限に抑えられています。
ベストプラクティスではなく、**境界**を定義しています。
